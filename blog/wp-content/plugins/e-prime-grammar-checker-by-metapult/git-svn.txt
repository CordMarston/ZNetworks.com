# GIT

Pushing Master back to SVN is done with this command:

	git svn dcommit

# SVN

Getting the latest trunk version in the SVN master repo

	svn update


Pushing changes in Trunk to the repo

	svn ci -m "comment"

Tagging a new Release

	svn cp trunk tags/1.0.1
	svn ci -m "comment"

# WP.ORG
